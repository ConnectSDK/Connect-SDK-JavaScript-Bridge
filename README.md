#Connect SDK JavaScript Bridge
Connect SDK is an open source framework that unifies device discovery and connectivity by providing one set of methods that work across multiple television platforms and protocols.

For more information, visit our [website](http://www.connectsdk.com/).

* [General information about Connect SDK](http://www.connectsdk.com/discover/)
* [Platform documentation & FAQs](http://www.connectsdk.com/docs/tv-web-app/)

##Including Connect SDK in your web app
* [Create a TV Web App](http://www.connectsdk.com/docs/tv-web-app/create-connected-web-app/)
* [Port a Receiver App to webOS](http://www.connectsdk.com/docs/tv-web-app/connect-my-web-app/)
